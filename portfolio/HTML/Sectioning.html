Basic html structure

Every HTML document starts with a doctype declaration, followed by the html tag, head tag, and body tag. The head contains metadata and the title of the page, while the body contains the content that is displayed in the browser.

Attributes in html

Attributes provide extra information about an html element.They tell the browser what to do,how to behave and what the element means

Common html elements

Some common html elements include headings (h1 to h6), paragraphs (p), links (a), images (img), lists (ul, ol, li), and tables (table, tr, td).
